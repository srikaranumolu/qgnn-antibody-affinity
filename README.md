# Quantum-Antibody-Binding

Repository skeleton for the RSEF project: quantum and classical GNNs for antibody binding prediction.

This repo contains a structured layout with placeholder files so contributors can jump in.

Structure (high level):
- data/: raw and processed data, splits, stats
- models/: classical and quantum model code and checkpoints
- scripts/: runnable scripts for the full pipeline
- src/: importable library code used by scripts and notebooks
- notebooks/: analysis and experiments
- results/: outputs from experiments
- tests/: basic unit tests
- docs/: documentation and notes
- configs/: YAML configuration files

See the individual directories for starter code and examples.
